# EventEase

EventEase app is a directory of upcoming career-focused events that enables people to register and keep track of their events.

How to run:

```powershell
dotnet restore
dotnet run
```

To display the blazor wasm app open the browser at `https://localhost:5001` (or as reported by the dev server).

For this project I had to use Microsoft's Copilot to help in the creation of components, debugging, and improvement of the application.

For this coursera course project I had to develop this application in three steps:
 - Main application and event card component.
 - Debugging and input validation.
 - Improvements and tests.

Copilot helped me in the Main application and event card component by:
 - Suggesting some design elements.
 - Suggesting responsive design principles applied to Blazor app development using Bootstrap and CSS.
 - Keeping a to-do list of appropriate improvements.

Copilot helped in the debuggin and input validation process by:
 - Suggesting basic best practices for handling input validation.
 - Suggesting tests for each possible action to validate correct functionality.

Copilot helped in the improvement of the application by:
 - Suggesting ways to create the UserSessionService to handle all data related to the appropriate user.

Because of the scope of this project being completely front-end, there is no login, nor more advanced and appropriate user handling.

For my daily work I use Angular for the front-end. I must say that I like the direction Blazor is taking and I look forward to working with it in the future.
